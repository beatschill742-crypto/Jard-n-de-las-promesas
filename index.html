``` ### El último paso obligatorio (La Regla de Oro) Para que Firebase te deje guardar los teléfonos sin que te marque "Error de conexión", tienes que hacer esto una sola vez: 1. Entra a [Firebase Console](https://console.firebase.google.com/). 2. Entra a tu proyecto `creador-de-pases-con-numero`. 3. En el menú de la izquierda, dale clic a **Firestore Database**. 4. Arriba, vas a ver unas pestañas que dicen "Datos" y **"Reglas"**. Dale clic a **"Reglas"**. 5. Vas a ver un código que dice algo como: `allow read, write: if request.time < timestamp.date(2026, 7, 16);` (O algo que dice `false`). 6. **Cámbialo para que diga:** ```text allow read, write: if true; ``` 7.
